Final Project Proposal


Product Name: Noteify


Team Members: Matthew Senese, Matthew O'Malley, Jon Rayo


Application Overview:

This app will allow a user to take down notes and search through them


Technology Overview:

This app will use MongoDB, Node, Express, GitHub, and Azure


Feature List:

  • Menu of buttons

Future Features:

  • Stylized and simple layout
  • Ability to create a note
  • Authentication system to personalize user notes

Basic Technical Features:

This app will include nicely styled menu buttons so that it is easy to navigate


Advanced Technical Features:

A system to send email/text messages to user as a reminder feature
