Final Project Proposal


Product Name: Noteify


Team Members: Matthew Senese, Matthew O'Malley, Jon Rayo


Application Overview:

This app will allow a user to create an account, take down notes, and search through notes saved to thier account


Technology Overview:

This app will use Firebase, Node, Express, GitHub, and Azure


Feature List:

  • Menu of buttons
  • A field to login to
  • A field to write a note with a title
  • Stylized and simple layout
  • Ability to save a note
  • Authentication system to personalize user notes

Future Features:

  • A page to view notes saved to a user's credentials
  • An option to delete an account

Basic Technical Features:

  • This app will include nicely styled menu buttons so that it is easy to navigate
  • A login and logout system

Advanced Technical Features:

  • A system to send email/text messages to user as a reminder feature
  • A way to remove and edit data in the database
